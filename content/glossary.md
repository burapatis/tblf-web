## คำศัพท์เฉพาะทางของโครงการ

| คำศัพท์ | ความหมายในบริบทของโครงการ |
|---|---|
| **TBLF** (Thai Balanced Leadership Framework) | กรอบแนวคิดภาวะผู้นำแบบไทยพอดี — กรอบบูรณาการเชิงกระบวนการ (process-oriented integrative framework) สำหรับการใช้ภาวะผู้นำในการบริหารการศึกษาภายใต้บริบทประเทศไทย |
| **พอดี** | ความเหมาะสมของการตัดสินใจที่เกิดจากการพิจารณาคุณค่า หลักฐาน ข้อจำกัด ความเสี่ยง และผลกระทบในบริบทหนึ่ง |
| **BRS-01** (Baseline Reference Standard) | มาตรฐานอ้างอิงหลักของกรอบแนวคิด — รุ่นปัจจุบันคือ Integrated Master Edition v0.3 สถานะ Canonical Research-ready Baseline |
| **ฐานกำกับ** (Governing Foundations: G1–G5) | หลักการที่กำกับทุกระยะของกระบวนการ ได้แก่ G1 จริยธรรม, G2 เป้าหมายทางการศึกษา, G3 หลักฐาน, G4 ความเสมอภาค, G5 คุณค่าสาธารณะ |
| **แนวคิดแกน** (Core Constructs: C1–C4) | กระบวนการที่ระบุสิ่งที่ภาวะผู้นำทำ ได้แก่ C1 การวินิจฉัยบริบท, C2 ดุลยพินิจอย่างรอบคอบ, C3 การกำหนดและเอื้อการดำเนินงาน, C4 การเรียนรู้และปรับตัว |
| **กลไกสนับสนุน** (Supporting Mechanisms: M1–M7) | เงื่อนไขหรือวิธีที่ช่วยให้แนวคิดแกนทำงานได้ดี ได้แก่ Respectful Voice, Relational Trust, Participation, Capacity Building, Empowerment, Accountability & Transparency, Collaboration |
| **ผลลัพธ์** (Outcomes: O1–O4) | ผลต่อผู้เรียน บุคลากร องค์การ และระบบ/สังคม ที่ใช้เป็นข้อมูลย้อนกลับ |
| **เงื่อนไขขอบเขต** (Boundary Conditions) | ปัจจัยที่ทำให้การทำงานของกรอบแตกต่างกันระหว่างบริบท เช่น กฎหมาย นโยบาย ทรัพยากร วัฒนธรรมองค์การ |
| **Human Expert Review (HER)** | การวิจารณ์กรอบโดยผู้ทรงคุณวุฒิที่เป็นมนุษย์บน Baseline เดียวกัน |
| **Field Validation** | การตรวจสอบความเข้าใจ ความเป็นไปได้ กลไก และผลของกรอบในบริบทจริง |
| **L + S + E + H** | รหัสแหล่งที่มาขององค์ประกอบ: Literature Review + Integrated Synthesis + AI-assisted Expert Review + Human Expert Review |
| **ดู คิด ทำ เรียนรู้** | ภาษาฉบับเข้าใจง่ายของแนวคิดแกน C1–C4 ใช้ในสื่อเผยแพร่ เช่น Executive Summary, Decision Card และ Casebook |
| **Decision Card** | เครื่องมือรายการตรวจสอบ 10 ข้อใน 4 ขั้น สำหรับใช้กรอบแนวคิด TBLF ประกอบการตัดสินใจจริง |
| **Living Casebook** | แนวทางพัฒนากรณีศึกษาอย่างต่อเนื่อง — นำประสบการณ์การใช้ TBLF จริง (โดยได้รับอนุญาตและปกปิดตัวตน) มาเผยแพร่เป็น Casebook รุ่นถัดไป |
| **TBLT** (Thai Balanced Leadership Theory) | ข้อเสนอเชิงทฤษฎีเบื้องต้นที่ยกระดับจาก TBLF หลังผ่านการตรวจสอบเชิงเนื้อหาและเชิงประจักษ์ |
| **R&D** (Research and Development) | กระบวนการวิจัยและพัฒนาอย่างเป็นระบบ ที่โครงการใช้เป็นวิธีหลักในการสร้างและตรวจสอบองค์ความรู้ |
| **Foundation Charter** | เอกสารอ้างอิงหลักของโครงการ กำหนดเจตนารมณ์ หลักการ และจุดยืนทางวิชาการ |
| **Project Charter** | เอกสารกำหนดขอบเขต วัตถุประสงค์ โครงสร้างงาน และการบริหารโครงการ |
| **Research Blueprint** | แผนแม่บทการวิจัย ("พิมพ์เขียวทางวิชาการ") ที่เชื่อมโยงเอกสารและผลผลิตทุกประเภทให้เป็นระบบเดียวกัน |
| **Work Package (WP)** | ชุดงานย่อยของโครงการ มีทั้งหมด 10 ชุด ครอบคลุมตั้งแต่การวิจัยจนถึงการติดตามประเมินผล |
| **Pragmatism** | ปรัชญาการวิจัยที่โครงการยึดถือ เน้นการผสมผสานวิธีวิจัยเชิงคุณภาพและเชิงปริมาณเพื่อให้องค์ความรู้ใช้ได้จริง |
| **Evidence-based** | หลักการยึดหลักฐานเชิงประจักษ์เป็นฐานของทุกข้อเสนอและข้อสรุป |
| **Knowledge Architecture** | สถาปัตยกรรมองค์ความรู้ 4 ชั้นของโครงการ: Foundations → Framework → Theory Proposition → Applications |
| **ความตรงเชิงเนื้อหา** (Content Validity) | คุณภาพของกรอบแนวคิด/เครื่องมือ ที่ได้รับการตรวจสอบว่าเนื้อหาครอบคลุมและเหมาะสม โดยผู้ทรงคุณวุฒิ |
| **การตรวจสอบเชิงประจักษ์** (Empirical Validation) | การทดสอบกรอบแนวคิด/แบบจำลองด้วยข้อมูลจริงจากภาคสนาม |
